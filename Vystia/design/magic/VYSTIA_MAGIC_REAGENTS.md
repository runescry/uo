# Vystia Magic Reagent System

**Purpose:** Each magic school uses unique Vystia-themed reagents instead of standard UO reagents.
**Last Updated:** 2025-12-05

---

## Reagent Design Philosophy

1. **Regional Theming:** Reagents match the region's environment
2. **Unique Identity:** Each school has 3-5 primary reagents + 1-2 rare reagents
3. **Thematic Consistency:** Reagent names reflect the magic school's theme
4. **Graduated Rarity:**
   - **Common (Circles 1-3):** Easy to obtain, 1-2 types
   - **Uncommon (Circles 4-6):** Moderate scarcity, 2-3 types
   - **Rare (Circles 7-8):** Hard to obtain, special locations only

## Reagent Item Template (ServUO)

```csharp
namespace Server.Items.VystiaReagents
{
    // Example: Frostbloom (Ice Magic, Circle 1-3)
    public class Frostbloom : BaseReagent
    {
        public override int LabelNumber { get { return 1032684; } } // Custom label

        [Constructable]
        public Frostbloom() : this(1)
        {
        }

        [Constructable]
        public Frostbloom(int amount) : base(0x0F88, amount) // Item graphic ID
        {
            Name = "Frostbloom";
            Hue = 0x481; // Ice blue hue
            Weight = 0.1;
        }

        public Frostbloom(Serial serial) : base(serial)
        {
        }

        public override void Serialize(GenericWriter writer)
        {
            base.Serialize(writer);
            writer.Write((int)0); // version
        }

        public override void Deserialize(GenericReader reader)
        {
            base.Deserialize(reader);
            int version = reader.ReadInt();
        }
    }
}
```

## Reagent Drop Locations

### By Region
- **Frosthold:** Ice Magic reagents
- **Emberlands:** Elemental Magic reagents
- **Verdantpeak:** Nature Magic reagents
- **Shadowfen:** Hex Magic reagents
- **ShadowVoid:** Dark Magic + Necromancy reagents

- **Deepwater Trench:** Summoning Magic reagents
- **Wilderlands:** Shamanic Magic reagents
- **Multi-Regional:** Bardic, Enchanting, Illusion reagents

### Drop Tables
**Creatures should drop reagents matching their region:**
- Common mobs (wolves, bears): Common reagents (Circles 1-3)
- Elite mobs (champions): Uncommon reagents (Circles 4-6)
- Bosses (regional lords): Rare reagents (Circles 7-8)
