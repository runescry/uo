## Best Practices

**For Dust/Powder Reagents:** Use **0x0F8F** (Grave Dust)
**For Herbs/Plants:** Use 0x0F84-0x0F88 range
**For Liquids/Vials:** Use 0x0F7D or 0x1C18
**For Crystals:** Use 0x0F8E or 0x0F7A

All confirmed stackable via `Stackable = true` in source code.
