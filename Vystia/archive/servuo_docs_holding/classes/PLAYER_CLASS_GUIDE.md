# Vystia Class System - Player Guide

Welcome to the Vystia Class System! This guide explains the 26 unique character classes available in the Vystia shard.

---

## Quick Start

1. **Choose a Class** - Use the class selection menu or ask a GM
2. **Learn Your Resource** - Each class has unique secondary resources
3. **Master Your Stances** - Some classes can switch between combat stances
4. **Use Your Abilities** - Classes have unique abilities tied to their spellbooks

---

## Class Categories

### Magic Classes (12)

Magic classes primarily use **Mana** and cast spells from their class-specific spellbooks.

| Class | Region | Secondary Resource | Specialty |
|-------|--------|-------------------|-----------|
| **Ice Mage** | Frosthold | Chill Stacks | Freeze enemies, shatter for burst damage |
| **Warlock** | ShadowVoid | Soul Shards (max 3) | Powerful dark magic, generate shards on kills |
| **Necromancer** | ShadowVoid | Life Force (max 100) | Raise undead, drain life from enemies |
| **Druid** | Verdantpeak | Forms | Shapeshift into Bear, Cat, Tree, or Moonkin |
| **Sorcerer** | Emberlands | Elemental Stances | Master Fire, Water, Earth, and Air magic |
| **Bard** | Multi-Regional | Crescendo (ticks) | Buff allies with songs, debuff enemies |
| **Witch** | Shadowfen | - | Hex magic, curses, and life drain |
| **Oracle** | Crystal Barrens | - | Divination, time manipulation, foresight |
| **Summoner** | Underwater | - | Summon powerful creatures to fight for you |
| **Shaman** | Multi-Regional | - | Spirit magic, totems, nature and ancestor spirits |
| **Enchanter** | Multi-Regional | - | Enhance weapons and armor, magical crafting |
| **Illusionist** | Desert | - | Create illusions, misdirection, stealth |

### Martial Classes (14)

Martial classes primarily use **Stamina** and excel in physical combat.

| Class | Region | Secondary Resource | Specialty |
|-------|--------|-------------------|-----------|
| **Barbarian** | Frosthold | Fury (max 100) | Enter Rage for massive damage, decay out of combat |
| **Rogue** | Multi-Regional | Combo Points (max 5) | Build combos, execute finishers, stealth |
| **Monk** | Ironclad | Chi (max 5) | Martial arts, no decay, three stances |
| **Knight** | Ironclad | Fortitude (max 10) | Tank, generate on blocks, protect allies |
| **Paladin** | Multi-Regional | Virtues (4 stacks) | Holy warrior, three auras, tithing |
| **Ranger** | Desert | Focus (max 100) | Ranged DPS, tracking, terrain mastery |
| **Fighter** | Ironclad | Stances | Versatile warrior with four combat stances |
| **Templar** | Ironclad | Zeal (max 10) | Holy knight, generate Zeal on attacks |
| **Bounty Hunter** | Multi-Regional | Pursuit (max 10) | Track and mark targets, execute bounties |
| **Beastmaster** | Frosthold | - | Tame and command powerful beasts |
| **Artificer** | Ironclad | Steam (100) + Charges (10) | Build gadgets, deploy turrets |
| **Alchemist** | Verdantpeak | - | Potions, transmutation, bombs |
| **Cleric** | Multi-Regional | Faith (max 100) | Healer, generate Faith from healing |
| **Wizard** | Crystal Barrens | - | Pure arcane magic, spell mastery |

## Secondary Resources Explained

### Combat-Generated Resources

| Resource | Class | How to Generate | How to Spend |
|----------|-------|-----------------|--------------|
| **Soul Shards** | Warlock | Kill enemies, critical hits | Powerful dark spells |
| **Life Force** | Necromancer | Deaths nearby | Raise undead, drain spells |
| **Fury** | Barbarian | Deal/take damage | Rage abilities, transforms |
| **Chi** | Monk | Deal damage | Martial techniques |
| **Combo Points** | Rogue | Basic attacks on target | Finisher abilities |
| **Fortitude** | Knight | Block attacks | Defensive abilities |
| **Zeal** | Templar | Deal damage | Holy strikes |
| **Pursuit** | Bounty Hunter | Attack marked targets | Execute bounties |
| **Faith** | Cleric | Heal allies | Divine abilities |

### Special Resources

| Resource | Class | Behavior |
|----------|-------|----------|
| **Chill Stacks** | Ice Mage | Build on enemies, 5 stacks = Frozen |
| **Crescendo** | Bard | Builds while channeling songs |
| **Steam** | Artificer | Regenerates near machinery |
| **Charges** | Artificer | Crafted, used for gadgets |
| **Virtues** | Paladin | 4 virtue stacks for abilities |

## Stance System

Some classes can switch between combat stances for different bonuses:

### Fighter Stances
- **Aggressive** - +Damage, -Defense
- **Defensive** - +Defense, -Damage
- **Balanced** - Moderate bonuses to both
- **Berserker** - Maximum offense, no defense

### Druid Forms
- **Bear Form** - Tank, +Armor, +HP
- **Cat Form** - DPS, +Crit, +Speed
- **Tree Form** - Healer, +Healing
- **Moonkin Form** - Caster, +Spell Power
- **Travel Form** - +Movement Speed

### Sorcerer Elements
- **Fire Stance** - Burst damage
- **Water Stance** - Healing, CC
- **Earth Stance** - Defense, shields
- **Air Stance** - Speed, evasion

### Monk Stances
- **Windwalker** - DPS, agility
- **Brewmaster** - Tank, mitigation
- **Mistweaver** - Healer, restoration

### Rogue Stances
- **Shadow** - Stealth, assassination
- **Outlaw** - Brawling, dirty tricks
- **Subtlety** - Control, precision

### Paladin Auras
- **Devotion** - Group defense
- **Retribution** - Group damage
- **Protection** - Tank bonuses

### Ranger Aspects
- **Hawk** - Ranged damage
- **Wolf** - Movement, pack tactics
- **Bear** - Survival, defense

## Tips for New Players

1. **Start with a Core Class** - Barbarian, Ice Mage, or Rogue are great for beginners
2. **Learn Your Resource** - Understanding your secondary resource is key to mastery
3. **Use Consumables** - Resource potions can turn the tide of battle
4. **Experiment with Stances** - If your class has stances, try them all
5. **Get Your Focus Item** - Visit the Class Equipment Dealer for your class focus item
6. **Read Your Spellbook** - Magic classes should familiarize themselves with all 32 spells
