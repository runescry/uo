## Getting Started

### Choosing Your Class

Vystia offers 26 distinct classes divided into two categories:

- **Magic Classes (12):** Spellcasters who use one of 12 magic schools
- **Martial Classes (14):** Combat specialists with unique ability sets

Each class is tied to a specific **region** of Vystia, granting access to regional equipment, resources, and lore.

### Class Selection

Use the class selection gump or speak to a Class Trainer NPC to choose your path. You can view class details before committing.

### Skills

Each class has a unique **class skill** (IDs 58-83) that governs their abilities:

| Skill Type | Skills |
|------------|--------|
| **Magic** | Cryomancy, Druidism, Hexcraft, Elementalism, Demonology, Divination, Necromancy Arts, Conjuration, Spirit Calling, Bardic Lore, Runeweaving, Illusion Magic |
| **Martial** | Berserking, Subterfuge, Martial Arts, Chivalric Arts, Holy Devotion, Marksmanship, Combat Mastery, Zealotry, Manhunting, Beast Bonding, Engineering, Transmutation, Divine Grace, Arcane Studies |

## Crafting Systems

Vystia features two custom crafting systems that use **Vystia skills** instead of standard UO skills:

### Transmutation (Alchemist)

**Skill Used:** Transmutation (ID 81)
**Tool:** Alchemist's Kit

Transmutation allows you to brew potions using Nature and Hex reagents from Verdantpeak and Shadowfen regions.

**Reagent Sources:**
- **Nature Reagents:** Wild Moss, Moonpetal, Druid Bark, Treant Sap, Primal Vine, Elderwood Seed, Living Bark, Ancient Root
- **Hex Reagents:** Bog Moss, Viper Fang, Swamp Lotus, Witchweed, Hag's Hair, Cursed Salt, Cursed Pearl, Toad's Eye

**Recipe Categories:**
| Category | Items | Skill Range |
|----------|-------|-------------|
| Healing Potions | Lesser/Normal/Greater Nature's Healing | 0-105 |
| Cure Potions | Lesser/Normal/Greater Antivenom | 0-115 |
| Refresh Potions | Stamina Tonic, Greater Stamina Tonic | 0-75 |
| Enhancement | Agility/Strength Elixirs, Night Vision | 15-95 |
| Poison Potions | Lesser/Normal/Greater/Deadly Venom | 0-140 |
| Explosive Potions | Lesser/Normal/Greater Explosive Flask | 5-115 |
| Special | Smoke Bomb | 90-120 |

### Engineering (Artificer)

**Skill Used:** Engineering (ID 80)
**Tool:** Engineering Tool Kit

Engineering allows you to craft clockwork components, gadgets, and mechanical equipment using Ironclad resources.

**Resource Sources:**
- **Primary:** Clockwork Ingot (smelted from Steamwork Ore)
- **Components:** Clockwork Gear, Clockwork Spring, Steam Core (crafted)

**Recipe Categories:**
| Category | Items | Skill Range |
|----------|-------|-------------|
| Basic Components | Clockwork Gear, Spring, Steam Core | 0-105 |
| Gadgets | Smoke Grenade, Small/Medium/Large Explosive | 25-130 |
| Tools | Engineering Tool Kit | 35-85 |
| Clockwork Items | Construct Control Device | 70-120 |
| Clockwork Equipment | Sword, Shield, Full Plate Set | 50-125 |

**Getting Started:**
1. Acquire Clockwork Ingots from mining/smelting or vendors
2. Craft Clockwork Gears first (low skill requirement)
3. Use gears to craft Springs and Steam Cores
4. Progress to gadgets and equipment

## Class Combos & Rotations

Master your class with these suggested ability combinations!

### Magic Class Combos

#### Ice Mage - "Shatter Combo"
```
Frost Touch → Freezing Grasp → Ice Bolt → Ice Bolt → Ice Bolt (5 Chill stacks)
→ Target FROZEN → Shatter (bonus damage vs frozen) → Blizzard (AoE cleanup)
```
**Key:** Build 5 Chill stacks to Freeze, then burst with Shatter.

#### Druid - "Bear Tank Rotation"
```
Wild Shape: Bear → Maul → Swipe (AoE threat) → Regeneration (self)
→ Entangling Roots (add control) → Maul → Nature's Wrath (burst)
```
**Key:** Stay in Bear Form for armor bonus. Use Roots to control adds.

#### Druid - "Cat DPS Rotation"
```
Wild Shape: Cat → Rake (DoT) → Shred → Shred → Shred
→ Ferocious Bite (finisher) → Rake (refresh) → repeat
```
**Key:** Keep Rake DoT up, spam Shred, finish with Bite.

#### Witch - "Curse Stack"
```
Bog Curse → Hex of Weakness → Poison Mist (AoE) → Death Bloom (root)
→ Swamp Familiar (summon) → Drain Life → refresh DoTs
```
**Key:** Stack multiple curses, let DoTs do the work while you drain.

#### Sorcerer - "Fire Burst"
```
Fire Stance → Magma Bolt → Magma Bolt → Volcanic Shield (reflect)
→ Pyroclastic Surge (point-blank AoE) → Elemental Embodiment (transform)
→ Magma Bolt spam during transform
```
**Key:** Use Volcanic Shield before Pyroclastic Surge for safety.

#### Warlock - "Soul Harvest"
```
Corruption (DoT) → Agony (DoT) → Eldritch Blast → Eldritch Blast
→ On Kill: Gain Soul Shard → Life Drain (heal) → Dark Pact (if low)
→ Summon Patron (spend 3 shards for big damage)
```
**Key:** Farm Soul Shards from kills, spend on Patron summon.

#### Oracle - "Fate Controller"
```
Glimpse Future (dodge buff) → Psychic Bolt → Fate Weave (crit buff ally)
→ Psychic Bolt → Crystal Communion (debuff enemy) → repeat
```
**Key:** Support allies with Fate Weave while dealing psychic damage.

#### Necromancer - "Army of the Dead"
```
Raise Undead → Raise Undead → Raise Undead (3 minions)
→ Life Drain (sustain) → Death Nova (AoE burst) → Raise more
→ Army of the Dead (ultimate - mass summon)
```
**Key:** Always maintain minions. Life Drain keeps you alive.

#### Summoner - "Elemental Assault"
```
Summon Fire Elemental → Summon Water Elemental → Binding Circle (control)
→ Direct elementals to attack → Summon Earth Elemental (tank)
→ Mass Summon (ultimate - army)
```
**Key:** Summon tank elemental first, then DPS elementals.

#### Shaman - "Totem Master"
```
Elemental Totem: Fire (damage) → Elemental Totem: Healing
→ Spirit Wolves (summon) → Healing Rain (AoE heal) → Lightning Bolt
→ Ancestral Guidance (channel spirits for burst)
```
**Key:** Place totems first, then support with heals and summons.

#### Bard - "Crescendo Builder"
```
Inspiring Song (buff allies, +Crescendo) → Sonic Blast → Inspiring Song
→ At 10 Crescendo: Symphony of War (empowered party buff)
→ Charm (CC high-value target) → Counter-performance (interrupt)
```
**Key:** Build Crescendo with songs, unleash Symphony at max.

#### Enchanter - "Sigil Trap"
```
Sigil of Protection (on self) → Enchant Weapon (buff ally weapon)
→ Runic Trap (place trap) → Lure enemy into trap
→ Dispel Magic (remove enemy buffs) → repeat traps
```
**Key:** Prepare the battlefield with sigils and traps.

#### Illusionist - "Mirror Master"

**Key:** Create copies for confusion, blind, then control.

## Tips for New Players

1. **Choose a class that fits your playstyle** - Tank (Knight, Fighter), Healer (Cleric, Druid), DPS (Barbarian, Sorcerer), Support (Bard, Enchanter)

2. **Learn your resource system** - Each class has unique mechanics. Barbarians build Fury, Rogues build Combo Points, etc.

3. **Use your stances** - Many classes have stances that change their playstyle. Experiment!

4. **Stock up on potions** - Regional potions provide significant bonuses. Visit vendors!

5. **Read spell/ability tooltips** - Understanding what each ability does is key to success.

6. **Group up!** - Vystia is designed for group content. Find a party!

7. **Master your combos** - Learn the ability chains above for maximum effectiveness!

8. **Watch your resources** - Don't spam abilities. Manage Fury/Chi/Combo Points wisely.
