#### Creating Custom Races

Dwarves were implemented using the following approach:

1. **Body Animation**: Extract human body animations (400/401), resize to 75% scale
2. **Body IDs**: Write resized animations to new body IDs (987/988)
3. **Equipment**: Create scaled equipment animations (IDs 909-919)
4. **Client Files**:
   - `anim.mul` / `anim.idx` - Patched with dwarf animations
   - `bodyconv.def` - Disable redirects for 914-919
   - `equipconv.def` - Equipment conversion rules
   - `mobtypes.txt` - Define 987/988 as HUMAN type

5. **ClassicUO Modification**: Add body IDs to `IsHuman` check in `AnimationsLoader.cs`

#### Dwarf Spawn Commands

```
[sd   - Spawn Dwarf (random gender)
[sdm  - Spawn Male Dwarf
[sdf  - Spawn Female Dwarf
```

## 10. Combat Formulas

### Melee Damage

```
FinalDamage = BaseDamage + (BaseDamage × TotalBonus)

TotalBonus = STR_Bonus + Anatomy_Bonus + Tactics_Bonus + WeaponDamage%

STR Bonus:      (STR × 0.30) / 5.0  = ~1% per 5 STR
Anatomy Bonus:  (Skill × 0.50) / 5.0 = ~1% per 2 skill
Tactics Bonus:  (Skill × 0.625) / 6.25 = ~1% per 1.6 skill
```

### Armor Reduction

```
Damage_After_Resist = Base × (100 - Resistance) / 100

Player Resist Range: -70% to +70%
Creature Resist Range: -100% to +100%
```

### Swing Timer

```
ticks = floor((WeaponSpeed × 4 - Stamina/30) × (100 / (100 + SpeedBonus)))
SwingTime = max(5, ticks) × 0.25 seconds

Minimum: 1.25 seconds | Speed Bonus Cap: 60%
```

### Spell Casting Time

```
BaseCastTime = (4 + SpellCircle) × 0.25 seconds
FinalCastTime = BaseCastTime - (FC × 0.25)

FC Cap: 2-4 depending on spell type
Minimum: 0.25 seconds
```

### Mana Regeneration

```
Mana_Per_Tick = (INT / 30.0) + (RegenMana / 10.0)
Per_Second = Mana_Per_Tick × 4

Tick Rate: 250ms (0.25 seconds)
```
