## Overview

Dark Magic channels demonic and shadow energies, specializing in fear, chaos, demon summoning, and sacrificial magic. Warlocks trade health for power and command demons to fight alongside them.

**Strengths:**
- Powerful demon summons
- Fear-based CC
- High damage through sacrifice
- Chaos effects (RNG damage/debuffs)
- Soul shard mechanics

**Weaknesses:**
- Self-damage from sacrifices
- Vulnerable to holy magic
- Demons can be killed/dispelled
- Chaotic (unreliable effects)

### 🔧 Custom Mechanics

**1. Soul Shard System**
```csharp
public static class SoulShardManager
{
    private static Dictionary<Mobile, int> m_Shards = new Dictionary<Mobile, int>();
    private const int MAX_SHARDS = 5;

    public static void AddShard(Mobile m)
    {
        if (!m_Shards.ContainsKey(m))
            m_Shards[m] = 0;
        m_Shards[m] = Math.Min(m_Shards[m] + 1, MAX_SHARDS);
    }

    public static bool ConsumeShard(Mobile m)
    {
        if (m_Shards.ContainsKey(m) && m_Shards[m] > 0)
        {
            m_Shards[m]--;
            return true;
        }
        return false;
    }
}

// Drain Soul spell: If target HP < 10%, kill and grant shard
if (target.Hits < target.HitsMax * 0.1)
{
    target.Kill();
    SoulShardManager.AddShard(Caster);
}
```
