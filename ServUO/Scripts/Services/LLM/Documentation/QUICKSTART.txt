================================
LLM NPC QUICK START GUIDE
================================

STEP 1: Choose Your LLM Provider
---------------------------------
OPTION A: OpenAI (Cloud, Paid)
1. Go to https://platform.openai.com/api-keys
2. Sign up or log in
3. Click "Create new secret key"
4. Copy the key (starts with "sk-")

OPTION B: Ollama (Local, Free)
1. Download from https://ollama.com
2. Install Ollama
3. Open terminal: ollama pull phi3:mini
4. Verify: ollama list


STEP 2: Add Your Configuration
-------------------------------
OPTION A: OpenAI
1. Start ServUO once (creates config file)
2. Edit: Config\LLM.cfg
3. Replace YOUR_API_KEY_HERE with your actual key
4. Save the file
5. Restart ServUO

OPTION B: Ollama
1. Start ServUO once (creates config file)
2. Edit: Config\LocalLLM.cfg
3. Verify endpoint: http://localhost:11434
4. Verify model: phi3:mini
5. Save and restart ServUO


STEP 3: Test It In-Game
------------------------
1. Log into the game with a GM account
2. Type: [spawnllmnpc
3. Click a category (e.g., "Town NPCs")
4. Click an NPC type (e.g., "Blacksmith")
5. Click on the ground to place the NPC
6. Walk near the NPC and say their name
7. Start talking!


EXAMPLE CONVERSATION:
---------------------
You: "Hello Gareth!"
Gareth the Blacksmith: "Greetings, traveler! Welcome to my forge. The name's Gareth. What brings ye to my shop today?"

You: "Tell me about your work"
Gareth: "Ah, the craft of forging! I've been working metal for nigh on twenty years. Each blade I make has a soul of its own, shaped by fire and hammer."

You: "Can I buy a sword?"
Gareth: "Aye, I have fine blades for sale! Let me show ye what I've got in stock."
*Opens vendor buy gump*

TIPS:
-----
- Stand within 10 tiles of NPC to talk
- Say NPC's name to start conversation
- Right-click NPC → "Reset" to clear history
- Conversations expire after 2 minutes silence
- OpenAI: ~$0.001 per conversation (very cheap)
- Ollama: Free but requires 8GB+ RAM

TROUBLESHOOTING:
----------------
NPC says "I cannot speak right now"?
  → API key not configured or invalid
  → Check Config\LLM.cfg or Config\LocalLLM.cfg
  → Restart server after adding key

NPC doesn't respond?
  → Stand within 10 tiles
  → Say the NPC's name first
  → Check console for errors

Ollama not working?
  → Verify: ollama list
  → Check model downloaded: ollama pull phi3:mini
  → Verify endpoint: http://localhost:11434

Slow responses?
  → OpenAI: 1-3 seconds is normal
  → Ollama phi3:mini: 1-2 seconds normal
  → Check network (OpenAI) or CPU (Ollama)

MEMORY SYSTEM (Optional):
--------------------------
NPCs can remember players across sessions!

SQLite is used automatically (no setup needed!):
- Database file: Data/llm_memories.db (auto-created)
- No external servers required
- Just enable in Config\LLMMemory.cfg

Check status: [MemoryStats
View memories: [ViewMemories (click NPC to view)
