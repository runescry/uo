# ServUO Scripts Documentation

This folder contains general analysis and documentation for ServUO services and systems.

## Documentation Files

### Service Analysis
- **[SERVICE_ANALYSIS.md](SERVICE_ANALYSIS.md)** - Comprehensive overview of all services in the ServUO Services folder, including what each service does, how it works, how to enable it, and how it interacts with other systems.

### MMO Appropriateness Analysis
- **[MMO_APPROPRIATENESS_ANALYSIS.md](MMO_APPROPRIATENESS_ANALYSIS.md)** - Identifies services and features that are inappropriate, problematic, or need attention for a serious, engaging MMO production environment.

### Service Status
- **[CURRENTLY_ENABLED_SERVICES.md](CURRENTLY_ENABLED_SERVICES.md)** - Current status of all services based on configuration files, showing what's enabled, disabled, and broken.

## Service-Specific Documentation

Service-specific documentation remains in their respective folders:

- **LLM Service**: `ServUO/Scripts/Services/LLM/Documentation/`
- **Spawner Commands**: `ServUO/Scripts/Commands/SPAWNER_BASELINE_DOCUMENTATION.md`

## Organization Philosophy

- **General Analysis Docs** → `ServUO/Scripts/Documentation/` (this folder)
- **Service-Specific Docs** → Co-located with their code (e.g., `Services/LLM/Documentation/`, `Commands/SPAWNER_BASELINE_DOCUMENTATION.md`)

This hybrid approach keeps documentation close to code while providing a central location for cross-cutting analysis and overview documents.
